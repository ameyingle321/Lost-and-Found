# Lost & Found Web Application

A full-stack web application to help users report and recover lost items efficiently.

![Screenshot](./screenshot.png)

## Features

- Report lost and found items
- User authentication
- Database integration for secure item management
- Modern responsive UI

## Tech Stack

- React with TypeScript
- Vite
- Node.js / Express (backend)
- MongoDB / Supabase
- Authentication

## Getting Started

```bash
npm install
npm run dev
```
